
This is a multi-threaded, recursive raytracer made by Daniel Olson.

To run (Windows only):

From 1 directory down of where we are, project root, do:
	<executeable> <scnfile>
	
	example:
	./x64/Release/Raytracer.exe Complex/foo.scn
	
	Many .scn files are available in Complex, InterestingScenes, SphereExamples, and Triangles
